# Approve
---
### Opis
Odobri sugestiju.
### Argumenti
`sID` - ID sugestije. - **Obavezno**\
`reason` - Razlog zašto je sugestija odobrena. - **Neobavezno**
### Upotreba
```
-approve <sID> [reason]
```
### Cooldown
`2 sekunde`
### Potrebna Dozvola
**Administrator** ili **[Menadžer Sugestija](administration/suggestionsmanager.md)**
