# Reject
---
### Opis
Odbij sugestiju.
### Argumenti
`sID` - ID sugestije. - **Obavezno**\
`reason` - Razlog zašto je sugestija odbijena. - **Neobavezno**
### Upotreba
```
-reject <sID> [reason]
```
### Cooldown
`2 sekunde`
### Potrebna Dozvola
**Administrator** ili **[Menadžer Sugestija](administration/suggestionsmanager.md)**
