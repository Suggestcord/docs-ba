# Consider
---
### Opis
Označite sugestija kao možda.
### Argumenti
`sID` - ID sugestije. - **Obavezno**\
`reason` - Razlog zašto je sugestija označena. - **Neobavezno**
### Upotreba
```
-consider <sID> [reason]
```
### Cooldown
`2 sekunde`
### Potrebna Dozvola
**Administrator** ili **[Menadžer Sugestija](administration/suggestionsmanager.md)**
